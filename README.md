# Econyms – The Ultimate Economics Acronym Study App 📚

Econyms is a **completely free and offline** Android app built specifically for Ugandan A‑Level Economics students.  
It turns every economics mnemonic into an interactive game, quiz, or flashcard—making revision addictive and effective.

## ✨ Features
- **6 built‑in games** (Matching, Flashcards, Hangman, Builder, Verdict, Letter Rush)
- **5,000+ quiz questions** covering all topics
- **39 complete lessons** with detailed breakdowns
- **In‑app updates** – never chase APKs again
- **Study streaks & reminders** to keep you consistent

## 🔒 Is This App Safe?
Yes! This app is **open‑source** – you can read every line of code right here in this repository.  
If you ever see a warning from Google Play Protect, it's because the app isn't distributed through the Play Store.  
You can **verify the APK yourself**:
1. Download the APK from the [Releases page](https://github.com/muggagaryan-arch/econyms-updates/releases).
2. Compare its SHA‑256 hash with the one listed in the release notes.
3. Build the app from source by following the instructions below.

## 🛠 Building from Source
1. Install [Flutter](https://flutter.dev) and [Android Studio](https://developer.android.com/studio).
2. Clone this repo: `git clone https://github.com/muggagaryan-arch/econyms-updates.git`
3. Run `flutter pub get` and `flutter build apk --release`.
4. The APK will be at `build/app/outputs/flutter-apk/app-release.apk`.
